# CampusUP College - исходники сайта

Состав:
- index.html - главная страница;
- courses.html - раздел «Курсы»;
- team.html - раздел «Команда»;
- contact.html - раздел «Связаться»;
- styles.css - адаптивная верстка, hover/active эффекты, анимация логотипа;
- script.js - мобильное меню и демонстрационная обработка формы;
- logo.svg - логотип проекта.

Публикация:
1. Создать репозиторий `campusup-college` на GitHub.
2. Загрузить файлы из этой папки в корень репозитория.
3. В Settings -> Pages выбрать ветку main и папку root.
4. После публикации заменить ссылку-заглушку в отчете на реальную ссылку.

Важно: iframe Google Forms/Яндекс.Формы можно вставить в contact.html вместо локальной формы.
